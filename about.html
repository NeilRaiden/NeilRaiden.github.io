About NeilRaiden.com

The purpose of this website is to offer tools for learning English, especially for building phonemic awareness.

  • Phonetic dictionaries (coming soon)
  • Phonetic spellcheck dictionaries (coming soon)
  • Phonetic keyboard layouts

This website is hosted on GitHub.com.
This website itself is a GitHub project: GitHub.com/NeilRaiden/NeilRaiden.github.io.
