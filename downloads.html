Installation instructions

Mac OS

Installation on Mac is the simplest of all:

  1. Double-click on the downloaded .dmg file — a window will pop up displaying two options, for example:
    • ESL.bundle
    • Keyboard Installer
  2. Double-click the Keyboard Installer icon, its' window will pop up. At the bottom it says: “Keyboard layout to install: Drag keyboard layout here to install”.
  3. Drag and drop the .bundle icon to the “Keyboard Installer” window, the sentence will change to “Keyboard layout to install: ESL.bundle”.
  4. Click one of the two options:
    • Install for current user (recommended)
    • Install for all users (Sierra or earlier)

Configuration:

  1. Go to:  → System settings → Keyboard.
  2. Find “Text Input” section. In the line “Input Sources” click “Edit” button.
  3. To have a keyboard switcher on the menu bar, turn on the switch next to “Show Input menu in menu bar”
  4. Under “All input sources” (left side) click the “+” icon.
  5. Select language “English” or “Others”.
  6. Select layout to install and click the “Add” button.

Usage:

  • ⌃ Space — press Control + Space to switch layouts - selection window is displayed on the screen.
  • ⌃ ⌥ Space — press Control + Option + Space to switch layouts.
